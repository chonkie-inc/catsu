# Mimie Examples

## Available Examples

| Example | Description |
|---------|-------------|
| [voyageai.py](voyageai.py) | Complete VoyageAI usage guide with basic, batch, async embeddings, tokenization, and error handling |

## Setup

1. Install Mimie:
```bash
pip install mimie
```

2. Set your API key:
```bash
export VOYAGE_API_KEY='your-api-key-here'
```

3. Run an example:
```bash
python examples/voyageai.py
```
